MolViz v0.1.0 by Adrien Pelfresne <adrien.pelfresne@gmail.com>
    A simple OpenGL molecular visualization, capable of reading protein data bank files.

    Usage: cargo run --release -- --file <FILE>
    Options:
  -f, --file <FILE>
  -h, --help         Print help
  -V, --version      Print version
